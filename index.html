NRK TV downloader

What's this for?

  • NRK downloader is a windows software(.NET Framework 2.0+ required) to download videos from http://tv.nrk.no/ where it uses AdobeHDS streaming boardcast system.

  • What's more, NRK downloader enables you to download xml subtitles and convert xml subtitles to videoplayer-readable srt(SubRip) format.
